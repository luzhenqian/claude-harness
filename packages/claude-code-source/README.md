# Claude Code Source — No Longer Available

Anthropic has requested that Claude Code source code not be hosted on GitHub. In compliance with their policy, this directory no longer contains a functional package or any redistributed source.

The original source was leaked via a `.map` file in Anthropic's npm registry on 2026-03-31. Anthropic has since taken steps to prevent further distribution, and public GitHub repositories hosting the code have been taken down.
